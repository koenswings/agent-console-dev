## Decision confirmed

Solid.js + Chrome Extension is the selected stack. Key reason: fine-grained signal-based
reactivity matches the Engine's Automerge streaming update model exactly — only affected
DOM nodes re-render on each small event.

Flutter eliminated: cannot produce a Chrome Extension (hard blocker).

Mobile path: PWA on same codebase, if needed later.

## Task context
